Savvy Paws

Savvy Paws is an online web application that allows a user to log in the information of their pets. Think about is a social media for your adorable pets. In this application, an authenticated user can create, delete and update a pet information. Add a picture for each pet and see all pets in the database. The application provides a logged-out and logged-in view in a one-page website format.

In this project I developed both the front-end and the back-end using mainly the following technologies:

  • HTML5
  • CSS3
  • Boostrap
  • Handlebars
  • Ruby on Rails
  • JavaScript
  • jQuery
  • Heroku
  • AJAX
  • Git
  • RESTful API
