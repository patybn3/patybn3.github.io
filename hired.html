Full Stack Web Application.

Hired

Hired is a full stack web application built usind Express API and React.js where users can create, edit, delete and view "profiles". The idea behind this application is to build a "reversed indeed" or "job search" website, were a user or employer can search for candidates for their current open positions using keywords.

Hired displays a list of all candidates on file, a list of the profiles created by a specific user, and the details of each profile. This is the version one of this project

Version 2 is currently under construction and will include file uploads using AWS which is already set up with the Express API used as the back-end of this project. Version two will also include a search bar allowing a user to find candidated by typing matching keywords.

The third version of this project will allow a user to contact a candidate using a contact form or chat. Once this step is completed this project will be done, unless if over achiver me decides to implement some other crazy feature.

This project was developed individually, both the front-end and the back-end were built using mainly the following technologies:

  • HTML5
  • CSS3
  • Styled Components
  • React.js
  • JavaScript
  • Node.js and Express.js
  • MongoDB
  • Axios
  • Heroku
  • Git
